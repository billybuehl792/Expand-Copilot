# Expand Copilot
A ScriptUI Panel for After effects with useful tools
* Apply and remove presets and text animations
* Position elements in comp
* Order elements in comp

## Installation
1. Clone repository to `/Applications/Adobe\ After\ Effects\ 202x/Scripts/ScriptUI\ Panels/`
2. Restart After Effects
3. Add panel to UI from `Window/Expand Copilot.jsx`
